Install Morsefy

Morsefy is a simple JavaScript library to encode and decode Morse that works both in browser and NodeJS environments. As well as a command line utility.

npm install morsefy

bower install morsefy

npm install -g morsefy to install as CLI utility.

or download Morsefy.js

Using Morsefy

Morsefy.Js

<script src="bower_components/morsefy/lib/Morsefy.js"></script>

//Decoding

<script>document.write(Morsefy.decode('-- --- .-. ... . ..-. -.--'));</script>

//Encoding

<script>document.write(Morsefy.encode('Morsefy'));</script>


Morsefy NodeJS Module

var morsefy = require('morsefy');

var encoded = morsefy.encode('This message will be encoded to Morse');

var decoded = morsefy.decode('-- --- .-. ... . ..-. -.--');

console.log(encoded, decoded);


CLI Utility

$ morsefy encode "this message will be encoded to morse"

$ morsefy decode "-- --- .-. ... . ..-. -.--"
